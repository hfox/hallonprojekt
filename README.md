# Hallonpajprojektet

Projektet fokuserar för tillfället på att få upp en korskompileringsmiljö
med RPi-kompatibel libc.

## Katalogstruktur

    bin    Binära filer, kortavbildningar och förkompilerade
           exempel.
    doc    Mer ingående text på varje delmoment.
    src    Källkodsfiler
